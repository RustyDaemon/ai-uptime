# Code of Conduct

This project aims to be welcoming, respectful, and useful for contributors of all experience levels.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions before escalating.
- Focus feedback on the code, design, or behavior, not the person.
- Share context clearly when reporting bugs, reviewing changes, or proposing ideas.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or sustained disruption
- Bad language or imagery in project spaces
- Publishing someone else's private information without permission

## Enforcement

Maintainers may remove comments, close discussions, reject contributions, or restrict participation when behavior harms the project or its contributors.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer through a non-public GitHub contact channel when possible. Please avoid using public issues for personal safety reports.
